Power at the Edge.

The native dashboard for running Local LLMs on Apple Silicon.
Performance monitoring, model management, and inference in one place.

Get Started View Source

Real-time Monitoring

Visualize NPU, GPU, and CPU usage instantly. Track memory pressure and thermal states optimized for M-series chips.

Model Management

Built-in aria2 downloader for high-speed model fetching. Organize GGUF files and manage storage effortlessly.

Smart Presets

One-click configurations for Llama 3, Mistral, and Gemma. Optimized context windows and GPU layers for your specific RAM.

Service Control

Start, stop, and restart llama.cpp server instances directly from the UI. View logs in real-time.

Model Browser

Explore and discover new models. Filter by quantization, size, and compatibility with your hardware.

Native Speed

Built on top of llama.cpp with Metal acceleration. Zero overhead, maximum tokens per second.
